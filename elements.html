Framkvæmd vefsíðu

Undirbúningur

Gerð þessarar vefsíðu er hluti af verkefni í áfanganum VÉL403G, sem Hafliði Ásgerisson kennir vorið 2021.

Fyrsta sem ég gerði var að horfa á myndbönd kennarans um HTML og GITHUB, en þau má finna hér. Ég hafði aldrei unnið í HTML áður eða notast við forrit eins og Git til þess að setja kóðann minn á Github. Þess vegna ákvað ég að kanna þetta aðeins betur og skoðaði meðal annars þetta myndband.

Ég hlóð forritinu Brackets niður en það notaði ég til þess að breyta kóðanum seinna meir. Hægt er að hlaða forritinu niður hér. Einnig hlóð ég niður forritinu Git. Það gerði ég hér. Á síðunni eru leiðbeiningar um hvernig á að hlaða niður forritinu fyrir mismunandi stýrikerfi. Ég fór inn í "Downloads" hlekkinn og valdi svo það stýrikerfi sem var í tölvunni minni. Ég hlóð niður forritinu git í gegnum Homebrew.

Github

Ég hafði lítið notað Github áður en átti þó aðgangi og hafði notast við síðuna í einhverjum hópverkefnum. Hér að neðan má sjá leiðbeiningar að því hvernig maður býr til aðgang á Github.

  1. Ferð inn á vefsíðu Github
  2. Í hægra horninu smellir þú svo á "Sign Up"
  3. Þá skráir þú inn notendanafn, tölvupóstfang og lykilorð fyrir aðganginn. Einnig þarftu að leysa létta þraut sem tryggir að þú sért manneskja sem ert að skrá þig.
  4. Nú er mögulegt að velja hvers konar vinnu maður stundar, hversu mikla reynslu af forritun maður heftur og hvers vegna þú ákvaðst að notast við Github.

Nú þegar aðgangurinn hefur verið settur upp er komið að því að búa til "Repository" fyrir verkefni. Það er hægt að gera í þessum skrefum:

  1. Þú smellir á græna "New" hnappinn sem er hægra megin við "Repositories".
  2. Veldu nafn fyrir þetta "Repository"
  3. Veldu hvort kóðinn á að vera "Public eða "Private" (í þessu verkefni valdi ég "Public").
  4. Smelltu á "Create repository" neðst á síðunni.
  5. Nú er hægt að velja um 4 leiðir til þess að setja upp kóðan í Github. Svona lítur sú síða út.

Vefsíðan

Nú var komið að því að velja þema fyrir heimasíðuna og valdi ég það template sem mér leist best á. Vegna reynsluleysi hjá mér þá hentaði afar vel að geta valið template og að geta unnið svo út frá því. síðunni. Í boði voru mörg flott template en fannst mér þetta henta mínum stíl best. Þetta template má finna hér og heitir Massively.

Ég hlóð þessu template niður og byrjaði á að skoða kóðan og breytti honum þannig að síðan myndi henta mér. Til þess að setja inn myndir setti ég myndirnar fyrst inn í möppuna "Images" sem var í möppunni og breytti kóðanum þannig að mögulegt væri að birta myndina á þeim stað sem ég vildi.

Leiðbeiningar voru á síðunni um hvaða skipanir birtu textan í italic textaformi. Svona var það birt á vefsíðunni: Kóðinn leit svo svona út:

Einnig voru svona leiðbeiningar fyrir uppsetningu lista, titla, skilgreiningu, takka, tilvitnana og mynda.

Tenging Github og Git

Þar sem ég hafði mjög takmarkaðan skilning á git og Github til að byrja með þá hafði ég upphaflega tengt git vitlaust við Github og lenti því í vandræðum í hvert einasta skipti sem ég ætlaði að uppfæra síðuna. Ég fékk villuna "Git error - Fatal: remote origin already exists" . Með hjálp Google tókst mér þó að laga þetta, en lausnina fann ég hér. Ég setti inn skipunina git remote set-url origin https://github.com/your/repository og sett inn https linkinn fyrir mitt repository inn í stað linksins sem er þarna. Því næst valdi ég efstu leiðina sem sýnt er á myndinni að ofan frá Github. Ég afritaði þann texta og þá loksins virkaði þetta.

Því næst fór ég inn í flipann "Settings" á Github og breytti "Source" úr "None" í "Main" og fékk þannig URL af síðunni minni.

Þegar ég svo uppfærði kóðan í tölvunni í Brackets fór ég svo í Terminal í möppunni með öllum kóðunum og skrifaði inn eftirfarandi skipunalínur. Hér má sjá hvernig ég hlóð aftur inn file-unum "generic.html", "index.html" og "elements.html".
